# Example for using gRPC with nestjs

- [gRPC](https://grpc.io/)
- [nestjs](https://nestjs.com/)

## Prerequisites

- Nodejs v12.x.x
- [Yarn](https://classic.yarnpkg.com/en/)

## Running this example

### Running the server

```
$ cd server
$ yarn install
$ yarn start
```

### Running the client

Open other terminal

```
$ cd client
$ yarn start
response { id: 1, name: 'John' }
```
